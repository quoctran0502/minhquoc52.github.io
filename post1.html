Bắt đầu từ một câu hỏi...

“Khi em gõ facebook.com vào trình duyệt, điều gì xảy ra?” Câu hỏi nhỏ mở ra cả một thế giới: DNS để phân giải tên miền, IP để định danh máy, TCP để đảm bảo truyền tải tin cậy…

2

Luồng Byte và Luồng Ký Tự

Trong Java, stream được chia làm hai nhóm chính:

Loại Luồng Lớp Chính Dữ Liệu Xử Lý Ứng Dụng
Luồng Byte InputStream, OutputStream Nhị phân (binary) Ảnh, video, file nén, dữ liệu socket TCP
Luồng Ký Tự Reader, Writer Văn bản (text) File .txt, JSON, HTML, UTF-8

Tóm lại

  • Dùng byte stream cho dữ liệu thô (ảnh, nhị phân).
  • Dùng character stream cho văn bản (có mã hóa ký tự).
3

Ví dụ: Đọc và ghi dữ liệu qua Socket

Ví dụ nhỏ về Echo Server — server nhận dữ liệu từ client rồi phản hồi lại:


// File: EchoServer.java
import java.io.*;
import java.net.*;

public class EchoServer {
  public static void main(String[] args) throws IOException {
    ServerSocket server = new ServerSocket(9000);
    System.out.println("Server đang chạy...");

    Socket socket = server.accept(); // Chờ client kết nối
    BufferedReader in = new BufferedReader(new InputStreamReader(socket.getInputStream()));
    PrintWriter out = new PrintWriter(socket.getOutputStream(), true);

    String msg = in.readLine();
    System.out.println("Nhận từ client: " + msg);
    out.println("Server nhận được: " + msg);

    socket.close();
    server.close();
  }
}

Giải thích

  • BufferedReader đọc văn bản từ input stream.
  • InputStreamReader chuyển byte → ký tự.
  • PrintWriter ghi văn bản ra socket.
4

Kết luận

Nắm tầng giao thức + thử nghiệm socket giúp ta “nhìn thấy” Internet vận hành. Nền tảng này rất hữu ích khi tiến lên web backend, API và bảo mật.
