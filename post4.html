Thiết Kế Giao Diện Web Với HTML & CSS

Năm 2 • 2023
1

Khung Semantic HTML

Dùng thẻ ngữ nghĩa giúp cấu trúc rõ ràng, hỗ trợ SEO và trợ năng (screen reader). Khung cơ bản:


<header class="site-header">...</header>
<main id="main">
  <section aria-labelledby="hero-title">...</section>
  <section aria-labelledby="features-title">...</section>
  <article>...</article>
</main>
<footer class="site-footer">...</footer>

Mẹo semantic

  • Dùng <nav aria-label="Điều hướng chính"> cho menu.
  • Gắn id vào heading và tham chiếu bằng aria-labelledby.
  • Chỉ một <h1> cho mỗi trang; phần còn lại h2–h3 theo cấp.
2

Layout với Flexbox & Grid

Flexbox để xếp hàng/dòng, Grid để chia khu vực lớn. Ví dụ layout 2 cột nội dung – sidebar:


/* Container chuẩn */
.container { max-width: 1120px; margin: 0 auto; padding: 0 20px; }

/* Grid 2 cột */
.grid {
  display: grid;
  grid-template-columns: 2fr 1fr;
  gap: 20px;
}

/* Nav dùng Flex */
.nav {
  display: flex; align-items: center; justify-content: space-between;
}

Chuyển mobile-first

  • Bắt đầu 1 cột; nâng dần lên 2–3 cột bằng media query.
  • Đặt gap thay vì margin lẻ tẻ để dễ kiểm soát khoảng cách.
3

Responsive Typography & Spacing

Dùng biến CSS + clamp() để font co giãn mượt mà theo viewport.


:root{
  --step--1: clamp(12px, 0.85vw, 14px);
  --step-0 : clamp(14px, 1.0vw, 16px);
  --step-1 : clamp(18px, 2.2vw, 24px);
  --step-2 : clamp(26px, 3.2vw, 36px);
}
body{ font-size: var(--step-0); }
h1{ font-size: var(--step-2); line-height: 1.1; }
h2{ font-size: var(--step-1); }

Khoảng cách

  • Chuẩn hóa bằng bộ thang khoảng cách (8, 12, 16, 20, 28…).
  • Dùng min()/max()/clamp() cho padding/margin quan trọng.
4

Component: Card & Button

Một card đơn giản + nút hành động theo style của theme hiện tại:


<article class="card">
  <img src="pics/sample.jpg" alt="Mô tả ảnh">
  <div class="p-16">
    <h3 class="title">Tiêu đề</h3>
    <p class="muted">Mô tả ngắn gọn nội dung card.</p>
    <div class="actions">
      <a class="btn primary" href="#">Xem thêm</a>
      <a class="btn ghost" href="#">Lưu</a>
    </div>
  </div>
</article>

/* CSS khớp với styles.css/post.css của bạn */
.card{
  background: var(--surface);
  border: 1px solid var(--border);
  border-radius: var(--radius);
  overflow: hidden;
  transition: transform .25s ease, box-shadow .25s ease;
}
.card:hover{ transform: translateY(-4px); box-shadow: var(--shadow-md); }
.btn{ display:inline-flex; align-items:center; gap:10px; padding:12px 18px; border-radius:12px; }
.btn.primary{ background: linear-gradient(135deg, var(--brand), var(--brand-2)); color:#fff; border:0; }
.btn.ghost{ background: transparent; border:2px solid var(--brand); color: var(--brand); }

Tip nhỏ

  • Thêm trạng thái :hover, :focus-visible nhất quán cho A11y.
  • Dùng aspect-ratio cho ảnh để tránh layout shift.
5

Checklist A11y & Hiệu năng

  • Ảnh có alt có nghĩa; icon trang trí dùng aria-hidden="true".
  • Độ tương phản màu văn bản ≥ 4.5:1 cho cỡ chữ thường.
  • Dùng loading="lazy" cho ảnh dưới màn hình đầu tiên.
  • Nhóm CSS quan trọng vào 1 file; defer/async JS không cần thiết lập tức.
6

Kết luận

HTML tạo cấu trúc, CSS tạo trải nghiệm. Bắt đầu từ semantic + layout rõ ràng, thêm responsive thông minh và component tái dùng — bạn sẽ có một UI hiện đại, mượt và dễ mở rộng.
